## Clone the repository:
```
git clone https://github.com/RajkumarGajbhiye/Inventory_management_app.git
```
## Env Variables

Create a .env file in the root and add the following

```
NODE_ENV = development
PORT = 2000
DATABASE_PASSWORD  
DATABASE_USERNAME 
DATABASE_CONNECTION_STRING 
JWT_SECRET_KEY
JWT_EXPIRY 

RAZORPAY_KEY_ID
RAZORPAY_KEY_SECRET
```

## Install Dependencies (frontend & backend)

```
# For frontend:
cd frontend
npm install

# For backend:
cd backend
npm install
```

## Run

```
# Run frontend only
cd frontend
npm run dev

# Run backend only
cd backend
npm start
```
